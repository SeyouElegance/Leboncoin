L'ENTREPRISE

11 ans après sa création, leboncoin est le 1er site de petites annonces en France et s’affirme comme l’un des media de proximité préféré des français. Il est un des sites leader sur le marché de l’automobile, de l’immobilier et de l’emploi et le plus grand vide-grenier de France, avec plus de 25M d'annonces. Son concept original autour de « la bonne affaire au coin de la rue » et sa simplicité d’utilisation sont plébiscités par des millions d’utilisateurs. Leboncoin est une filiale du groupe Norvégien Schibsted.

DESCRIPTIF DE L'OFFRE

**Leboncoin : Do you React ? ** Vous êtes développeur passionné des technologies frontend ? Vous avez le goût des communautés opensource, des best practices, des nouveautés, des conférences sur les technologies du web ? Et vous êtes en plus rigoureux, capable de structurer et de factoriser correctement votre code ?
We want you !
Dans le cadre de son virage vers les technologies web modernes, Leboncoin s’est doté d’un groupe de travail de convergence web et se lance dans une migration vers une toute nouvelle architecture front :

  • Javascript ES6
  • React Js
  • Redux
  • Tests unitaires couverts à 100%
  • Package sur mesure, combinant webpack et d’autres…
  • Code Review et Pair Programming
  • Code versionning avec Github
  • Continuous Integration avec Travis
  • Méthode Scrum/Agile
Notre objectif :
  • Une approche API centrique
  • L’isomorphisme / Server Side Rendering (Node.js)
  • Une expérience enrichie sur desktop et web mobile

Venez rejoindre une équipe jeune, dynamique et humaine, forgée dans le partage et l’échange de compétences. Au sein de Leboncoin, élue Great Place To Work depuis de nombreuses années, l’ambiance est chaleureuse, fun, respectueuse et détendue…et il nous tient à cœur qu’elle le reste !

Si vous êtes tentés de rejoindre notre aventure, n’hésitez pas à nous faire part de votre candidature.

Pas de diplôme requis, la passion et la compétence sont suffisants.

Lieu : Paris Disponibilité : Dès que possible. Rémunération : Selon profil ; Rattachement hiérarchique : Responsable des Développements Frontend Web
